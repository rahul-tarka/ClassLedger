# ClassLedger Logo Assets

## Files

- **logo.svg** - Main logo for light backgrounds (homepage, login, teacher dashboard)
- **logo-white.svg** - White version for dark/gradient headers (admin, principal dashboards)
- **favicon.svg** - Favicon for browser tabs

## Logo Design

The logo features:
- **Book/Ledger Icon** - Represents attendance tracking
- **Checkmark** - Symbolizes attendance marking
- **Ledger Lines** - Shows the record-keeping aspect
- **Purple/Blue Gradient** - Professional, modern color scheme
- **"ClassLedger" Text** - Clear, bold typography
- **"by Tarka" Tagline** - Brand identification

## Usage

The logos are automatically used in:
- All page headers
- Browser favicon
- Homepage hero section

## Customization

To change colors, edit the SVG files and update the gradient definitions:
- Primary gradient: `#667eea` to `#764ba2` (Purple/Blue)
- Success color: `#10b981` (Green for checkmark)

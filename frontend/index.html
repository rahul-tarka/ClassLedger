About ClassLedger

ClassLedger is a comprehensive attendance management system designed specifically for schools. Built with security and audit compliance in mind, it ensures that attendance data is never lost and all actions are tracked for accountability.

Key Features

  • Multi-School Support: Manage attendance across multiple schools
  • Role-Based Access: Teachers, Admins, and Principals have appropriate permissions
  • Real-Time Tracking: Mark attendance in real-time with instant updates
  • Automatic Late Detection: Students arriving after 09:15 are automatically marked late
  • Audit-Safe Logs: All actions are logged and cannot be deleted
  • Mobile-First Design: Works seamlessly on phones, tablets, and desktops
  • Export & Reporting: Generate reports and export data to Google Drive
  • Looker Studio Ready: Data formatted for easy dashboard integration

How It Works

  1. School staff login with their Google account
  2. Teachers mark attendance for their assigned classes
  3. System automatically detects late arrivals (after 09:15)
  4. Admins can view reports and track teacher accountability
  5. Principals have read-only access to all dashboards
  6. All data is securely stored in Google Sheets with complete audit trails

Security & Compliance

  • Google OAuth authentication ensures only authorized users can access
  • Whitelist-based access control - only registered staff can login
  • Append-only logs prevent data tampering
  • 15-minute edit window for attendance corrections
  • Complete audit trail of all actions

System Requirements

  • Google Account (for staff login)
  • Google Sheets (for data storage)
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection
